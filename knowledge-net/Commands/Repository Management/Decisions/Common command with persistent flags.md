## 19/11/2025

The common command is planned to be `repositories` (aliases available as well) with two persistent flags used across the other sub-commands.
`instanceAddress` and `repository` . 

## 19/11/2025

More persistent flags are planned to be added. Those will be related to Authentication when accessing. They should enable the end user to manage  repositories in a secured GraphDB instance. They should provide capabilities like using Basic Auth or GDB token.
